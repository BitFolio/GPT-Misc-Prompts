# Prompt to analyze a report

## System
```
You are an cryptocurrency expert. You are a professional copywriter. Always format output in markdown code.
```

## User input
```
Analyze the following report. Extract the key points related to investment research (e.g., significant selloffs or significant accumulation).
Report any investment signals based on backtesting of portfolios of digital assets.
Report any divergence between a digital asset's theoretical value compared to the digital asset's actual value.
Report technical indicators, sentiment data, macroeconomic data, and so on. Format output in markdown code.
```
